# Expense Tracker Application

A modern, glassmorphism-styled expense tracker built with vanilla HTML, CSS, and JavaScript.

## Features
- **Add Expenses**: Track your spending with categories, dates, and amounts.
- **Visual Dashboard**: See your total balance and spending distribution at a glance.
- **Data Persistence**: Expenses are saved to your browser's Local Storage, so they remain after refreshing.
- **Delete Items**: Hover over an item to delete it.
- **Responsive Design**: Works on desktop and mobile.

## How to Run
Simply open `index.html` in your web browser.

## Customization
- Modify `styles.css` to change the color palette (variables at the top).
- Edit `app.js` to add or remove categories.
